## API

### Attributes

| 名称             | 说明                             | 类型   | 默认值 |
| ---------------- | -------------------------------- | ------ | ------ |
| target           | 触发滚动的对象                   | string | —      |
| visibilityHeight | 滚动高度达到此参数值才出现       | number | 200    |
| right            | 控制其显示位置，距离页面右边距   | number | 40     |
| bottom           | 控制其显示位置，距离页面底部距离 | number | 40     |

### Events

| 名称    | 说明               | 回调参数        |
| ------- | ------------------ | --------------- |
| onClick | 点击按钮触发的事件 | (event) => void |
