## API

### Attributes

| 属性名    | 说明                                                   | 类型                    | 默认值 |
| --------- | ------------------------------------------------------ | ----------------------- | ------ |
| open      | 是否显示 Dialog                                        | boolean                 | false  |
| title     | Dialog 对话框 Dialog 的标题                            | string &#124; ReactNode | ''     |
| width     | 对话框的宽度，默认值为 50%                             | string &#124; number    | ''     |
| showClose | 是否显示关闭按钮                                       | boolean                 | true   |
| footer    | footer                                                 | ReactNode               | —      |
| onClose   | 点击遮罩层或右上角叉的回调                             | () => void              | —      |
| header    | 对话框标题的内容；会替换标题部分，但不会移除关闭按钮。 | ReactNode               | —      |
