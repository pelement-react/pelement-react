## API

### Attributes

| 属性名   | 说明                  | 类型                                                 | 默认值 |
| -------- | --------------------- | ---------------------------------------------------- | ------ |
| icon     | result 组件的图标类型 | 'success' / 'warning' / 'info' / 'error' / ReactNode | info   |
| title    | result 组件的标题     | `string` / `ReactNode`                               | ''     |
| subTitle | result 组件的副标题   | `string` / `ReactNode`                               | ''     |
| extra    | 内容额外区域的内容    | `ReactNode`                                          | —      |
